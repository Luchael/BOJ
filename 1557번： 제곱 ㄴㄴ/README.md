## 문제

[문제 링크](https://boj.kr/1557)


<p>어떤수 N이 1이 아닌 제곱수로 나누어지지 않을 때, 이 수를 제곱ㄴㄴ수라고 한다. 제곱수는 4, 9, 16, 25와 같은 것이고, 제곱ㄴㄴ수는 1, 2, 3, 5, 6, 7, 10, 11, 13, ...과 같은 수이다.</p>

<p>K가 주어졌을 때, K번째 제곱ㄴㄴ수를 구하는 프로그램을 작성하시오.</p>



## 입력


<p>첫째 줄에 K가 주어진다.</p>



## 출력


<p>첫째 줄에 K번째 제곱ㄴㄴ수를 출력한다.</p>
